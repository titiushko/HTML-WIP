Bienvenidos a la página de Vicious


En esta página podrás encontrar un MANUAL de HTML completo con toda la documentación necesaria para poder publicar tu mismo/a tu propia página personal. Este manual es el "trabajo" de recopilación de varios manuales y referencias publicadas por World Wide Web Consortium (WC3), así como de libros que actualmente se hayan en el mercado de las distintas versiones de HyperText Markup Language.

Debido a mi cierta "fobia" por los productos Microsoft y lo que ello representa... este manual está enfocado hacia el HTML específico para Netscape, así como el manual de Javascript que estoy traduciendo de la versión inglesa para el mismo navegador. Espero que si eres un usuario de Internet Explorer, al menos encuentres algunos de los capítulos útiles, ya que salvo donde lo indique, por lo general soportan todas las DIRECTIVAS y ATRIBUTOS ambos navegadores.

Ah!! se me olvidaba!!... Si eres uno de esos usuarios que están dandole de comer a Telefónica, tienes una sección para que bajes la página completa en formato ZIP para que la puedas ver la OffLine sin estar conectado.

Espero que disfrutes del manual y que te sea útil para referencias si eres un desarrollador de Web's avanzado, o como introducción si eres un novato en esto del diseño WEB.
