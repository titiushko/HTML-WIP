Graficos e Iconos

Para incluir un Icono o un grafico en su pagina, se utiliza el comando IMG, el cual lleva un parametro SRC al cual se le asigna el nombre de la imagen o grafico.
la sintxis de este comando es parecida al de direccionar páginas.

Este comando no necesita de un comando de terminacion.
Un ejemplo de este comando es:

< IMG SRC="imagen.gif" >

Donde IMG es el comando y viene de IMaGe y SRC es el parametro y viene de SouRCe, la palabra imagen.gif corresponde al nombre del archivo que contiene una imagen, en este caso es una rchivo tipo GIF de COMPUSERVE.
Los Browsers de Internet soportan algunos tipos de Archivos de Imagenes, entre ellos se destacan los GIF's y los JPG.

  • *.gif, es un acronimo de General Image Format. Este formato fue inventado por Compuserve, tiene un formato simple y por eso es el mas popular en la Red.

  • *.jpg, es el acronimo de Journalist Photographic Equipment Group.
    En los MAC se usa *.jpeg y en los IBM *.jpg, ya que el MAC soporta extensiones con mas de 3 letras.
    Este es un formato comprimido, y solo se expande cuando se muestra, es decir;
    Si la Imagen JPG tiene un tamaño de 5KB cuando se muestra, tendra 1KB en el Archivo.

Si desea incluir una Imagen del tipo JPG, tenga en cuenta que debe escribir la extensión del archivo.
Por ejemplo; "image.jpg".

Animaciones.

Las animaciones son archivos con secuencias de imagenes, el formato general es *.gif.
En la Red hay varios paquetes de Software que generan estos archivos.

Tambien en la red encontrará una gran cantidad de animaciones, las cuales puede usar en su propia pagina.
El comando para colocar una animación es el mismo para colocar una imagen o un grafico, la diferencia la hace, el tipo de archivo y su contenido.

Papel Tapiz.

El papel tapiz es una imagen que se coloca para llenar el fondo de la pantalla.
Generalmente se usa un de formato JPG y el browser lo repite por toda la página.

El comando que se usa es BODY con el parametro Background, el comando BODY, como recordará, es uno de la estructura basica de una pagina, y contiene el cuerpo de lo que vamos a mostrar.
La sintaxis del comando es;

< BODY Background="image.jpg" >
