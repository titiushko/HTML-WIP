Direccionando Páginas

Para llamar otra página, sea de su propia aplicación o de cualquier otro lugar en la red, se utiliza el comando < a >
con el parametro "HREF", ejemplo:

< a href="http://www.geocities.com" >Ir a Geocities < /a >

este comando se verá así;

Ir a Geocities

El parametro href (Hypertext Reference) contiene la dirección completa de la página a la cual queremos hacer referencia.
Cuando el browser encuentra este comando Subraya el contenido entre y ,
Con solo colocar el puntero del Mouse, sobre la palabra subrayada ud podrá ver en la barra inferior, la dirección a la que está apuntando. Haga la prueba!.
Los ejemplos ud los ha observado en todo este curso.

Direccionando Sobre la Misma Página.

Así como el comado < a > le permite llamar a otra página en la red, este mismo comando le permite llamar a una dirección especifica dentro de la misma página, por ejemplo, cuando su home page es demasiado grande y desea ir al final de la pagina, lo puede hacer con solo presionar sobre una etiqueta, así: Ir al Inicio de la Leccion #4 .

Este comando se utiliza en dos partes;

  • Primero, debe especificar una etiqueta en una zona determinada, el comando es el siguiente;
    < a name="Etiqueta" >, donde name es el parámetro y etiqueta el nombre que ud quiera llamar.

  • Segundo, El comando para llamar a la etiqueta es el mismo para llamar una direccion, pero se antepone el simbolo #,así: < a href="#Etiqueta" >Ir al Inicio < /a >

Ahora está ud en capacidad de hacer un indice de Temas y relacionar cada tema con una etiqueta en particular.

Direccionando a un Correo Electrónico (e-mail).

Así como se puede direccionar una página en la red, Ud puede direccionar una Correo electróncio,
esto hace que sea más facil enviar un correo a una persona en aprticular, en este caso a Mi.

El comando que se usa es el siguiente;

< a href"mailto:jberdugo@metrotel.net.co" > Escribame. < /a >

Como puede ver Ud. es el mismo comando para direccionar, lo que cambia es el valor del parametro, en vez de colocar la dirección, coloca la palabra mailto: y la dirección del correo electrónico, el resultado del comando anterior será así:

Escribame
