Resumen Lección Antereior

En la lección anterior aprendío a guardar sus documentos en formato TEXTO usando el comando Guardar Como... de su editor de texto, además a ponerle la extensión a los archivos como *.htm para Windows 3.1 y *.html en MAC y Windows 95.
No debe insertar comandos de control en su documento, el HTML es el "lenguaje" de control para su documento.

Estructura de Comandos del HTML

La estructura de los comandos del HTML es muy simple, todos los comandos del HTML van encerrados entre los simbolos; menor que "<" y mayor que ">" así: < Comando >.
Aprender HTML es aprender los comandos que van encerrados entre estos simbolos y utilizarlos donde ud desee.

El objetivo del HTML es dar formato al texto de su página, estos formatos son Negritas, Italicas,Letra pequeña, etc. Para que el HTML cambie el efecto de un texto, por ejemplo a negritas, debe colocar el comando justo antes del texto que desea formatear y al final, colocar el cierre de comando, veamos esto con un ejemplo:

Este parrafo está en Negritas, y este No.

El comando que hace este efecto es la "b" de Negritas (Bold en Inglés).
Y la sintaxis es la siguiente:

Este parrafo está en Negritas, y este No.

Como podrá observar el comando "b" está encerrado entre < > al inicio del parrafo y al final se encuentra el mismo comando pero con un "SLASH" (/) que indica la terminación del comando.

Resumen de Comandos de Formato HTML
Comando Sintaxis Efecto
Negritas < b >Negritas< /b > Negritas
Italicas < i >Italicas< /i > Italicas
Texto Pequeño < tt >T. Pequeño< /tt > T. Peuqeño
    • Todos los comandos del HTML van encerrados entre < > sin exepción.
    • Los comandos se terminan con una sintaxis igual pero adicionando un slash antes del Comando.
    • Algunos comandos del HTML no necesitan del comando de cierre.
    • El HTML no distingue entre mayusculas y minusculas.
    • Los comandos del HTML pueden ir anidados o sea se pueden usar mas de un comando por texto a formatear.
    • Los comandos del HTML no se ven en las páginas, solo se vé el efecto de su uso.

Formateando su Propia Página

    Para Formatear su pagina con el HTML debe seguir estos pasos.

    • El Documento debe empezar con el comando HTML así:
      Esto indica al Browser que la página tiene comandos del HTML y que la trate como tál.
    • El siguiente comando debe ser: y
      Este indica el titulo de su página, si observa la parte superior de su browser encontrará a lo que me refiero.
    • Después viene el contenido del documento encerrado por: y
      Este comando indica el cuerpo del documento.
    • Y por ultimo el comando .
      Que es final del comando html, y por tanto el final del documento.
Lo mejor en este momento es hacer un ejemplo de lo anterior:

< html >

< title > Mi primera Home page < /title >

< body >

< b > Esta es mi primera página < /b > < p >

Los formatos son < i > Italicas < /i > y < b > Negritas < /b > < br >

< b > < i > o puedo usarlos combinados Negritas e Italicas < /i > < /b > < br >

< tt > y si quieres en letra pequeña.. < /tt >

< /body >

< /html >

En el ejemplo anterior encontrará 2 comandos nuevos, como recordará, si guardamos nuestro documento en formato texto y solo texto los caracteres de control no se guardarán, o sea que los retorno de carro o parrafo o línea, tampoco se guardarán, porque estos tambien son caracteres de control.
Es por esto que el HTML viene con los comandos < p > y < br > que insertan una línea en blanco y un salto de línea.

Note ud. que estos comandos no utilizan el comando de Cierre.
El comando < p > viene de la palabra "paragraph" parrafo y trata al texto como si fuera un parrafo, este comando se puede usar sin el comando de cierre o con él.

El comando < br > viene de la palabra "Break", el cual termina una línea de texto y salta a la línea siguiente sin dejar espacios.

Se fijó bien en los comandos < html > , < body > y < title >.
También Observó como los comandos < p > y < br >, terminan las líneas y lo envían a la siguiente?.
