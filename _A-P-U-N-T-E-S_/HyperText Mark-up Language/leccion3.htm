Resumen Lección Antereior

En la lección anterior aprendío a dar formato con los comandos Negrita, Italica y Letra Pequeña.
Que los comandos del HTML van encerrados entre < > sin excepción.
Además que los comandos encierran el parrafo o las palabras a las que le vamos a dar formato.
Que algunos comandos no necesitan del cierre de comandos, asi como el < br > y el < p >.
Ahora veremos algunos otros comandos que afectan la presentación de los textos.

Comandos para Encabezados de Texto.

Los comandos para encabezados se usan en la mayoría de paginas. Los hay de diferentes tamaños, (6) en total y su estructura es muy facil, por ejemplo:

< h1 > hasta < h6 >, donde el < h1 > es el encabezado más grande y el < h6 > es el más pequeño.

Los enacbezados en Html se generan en Negritas y tambien se le puede aplicar el formato de Italicas, el uso es mu sencillo, solo tiene que colocar el comando con la letra "h#" con el numero de encabezado. Los encabezados siempre llevan consigo un comando < p > ,por eso dejan una líne en blanco debajo de ellos.

< h1 >Este es el mas grande < /h1 >

< h3 >Este es el Mediano < /h3 >

< h6 >Este es el más pequeño < /h6 >

Comandos para Cambiar tamaño de Texto

Así como hay tamaños para los encabezados Ud. puede cambiar el tamaño del texto que ud desee, para eso debe usar el comando < Font Size >, para ello cuenta con (12) doce tamaños diferentes desde el -1 hasta el +6.
Lo anterior se conoce como tamaños relativos, donde el -6 es el más pequeño y el +6 es el más grande.
El comando < Font Size > se cierra con el comando < /Font > al cual no se incluye el Size. Veamos un ejemplo:

< Font Size="+3" >Tamaño="+3"< /font >

< Font Size="+1" >Tamaño="+1"< /font >

< Font Size="-1" >Tamaño="-1"< /font >

< Font Size="-3" >Tamaño="-3"< /font >

Que observó del ejemplo anterior?. Algunos comandos del HTML tiene argumentos.
Los argumentos son comandos dentro del comando, el cual imparte una caracteristica adicional al comando que lo lleva, asi como el comoando < Font > lleva el argumento "size=", hay otros comandos del HTML que llevan otros tipos de argumentos, los cuales veremos más adelante.

Comandos para Cambiar Tipo de Texto.

Los comandos para cambiar el tipo de texto, depende de los tipos de letra instaladas en su sistema, por ejemplo, en windows hay tipos de letra que se llaman "ARIAL", "COURIER", etc.
Estos tipos de letra es a lo que me refiero, ud puede cambiar el tipo de Texto a sus documentos con el comando "Face=", que es un argumento de < Font >.

< Font Face="ARIAL" >Tipo Arial < /Font >

< Font Face="COURIER" >Tipo Courier < /Font >

< Font Face="SYMBOL" >Tipo Symbol abcdefgh

< Font Face="TIMES" >Tipo Times < /Font >

< Font Face="HELVET" >Tipo Helvetica < /Font >

< Font Face="ROMAN" >Tipo Roman < /Font >

Los valores pasados a los argumentos deben estar encerrados entre comillas (" ").
Los comados para formatear texto se pueden combinar para dar una mejor presentación a su documento.

Comandos para Centrar Texto.

Habrá notado ud. que todos los ejemplos que hemos usado están centrados?
Pués bien, el HTML tambien incluye comandos para la alineación de Textos.
El comando para centrar textos es el < Center > y < /center >, el siguiente es un ejemplo de los anterior:

< Center >

Este texto esta Centrado

< /Center >

Por defecto todos los Browser alinean los textos a la izquierda.
Para alinearlos a la derecha se usa el comando < p Align="Right" > y < /p >.
El Argumento "ALIGN" puede tomar varios valores a saber:
Left, Para justificar a la Izquierda (por Defecto)
Right, Para Justificar a la derecha y,
Center, para Justificar al centro.
Veamos un ejemplo:

< p align="left" >Justificado a la Izquierda. < /p >

< p align="Center" >Justificado al centro.< /p >

< p align="Right" >Justificado a la derecha.< /p >
