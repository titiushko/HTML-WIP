Descripción extensa para la ilustración del pixel de referencia vs. el pixel del dispositivo

Este diagrama ilustra la relación entre el pixel de referencia y los pixeles del dispositivo (llamados "puntos" abajo).

La imagen muestra la salida de una impresora laser de alta resolución (con gran densidad de puntos) a la izquierda, y un monitor de pantalla de baja resolución a la derecha. Para la impresora laser, un pixel de referencia cuadrado es implementado por 16 puntos. Para el monitor de pantalla, un pixel de referencia es implementado por un único punto.
