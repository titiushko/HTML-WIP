Este diagrama ilustra cómo las cajas de bloque anónimas pueden aparecer alrededor del contenido anónimo.

Muestra dos cajas rectangulares que contienen texto, siendo el primero anónimo. Tiene un fondo gris claro y contiene las palabras "Un texto". Justo abajo hay una caja del elemento P con un fondo gris oscuro que contiene las palabras "Más texto". Las dos cajas están dentro de una caja DIV.
