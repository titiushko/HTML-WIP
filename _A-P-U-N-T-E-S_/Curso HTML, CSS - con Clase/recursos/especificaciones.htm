Traducciones de las especificaciones del W3C al castellano

Varias personas y organizaciones se han ofrecido voluntarias para traducir al castellano algunas de las recomendaciones del W3C. La mayor recompensa para los que realizamos esta labor es que estas traducciones resulten ser útiles para los diseñadores web hispanohablantes, y que las páginas web hechas por todos nosotros sean cada vez mejores, más compatibles y más accesibles.

A continuación mostramos la relación de las especificaciones traducidas. Recordad que las únicas versiones normativas son las versiones originales en inglés, que se pueden encontrar en el sitio web del W3C, y cuyas URLs se incluyen en la relación siguiente.
