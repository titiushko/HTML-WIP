Tutoriales en castellano

Probablemente uno de los mejores y más completos tutoriales de HTML y creación de páginas web (entre otras muchas cosas) en la red es el del estudio de webaprendiz, escrito por Ferran Martí.

Visita obligada a la Tierra de Nómadas, creación de Sysifus.

También os podemos recomendar webestilo.com. Tienen un manual sencillo de HTML con foros para preguntar, un Manual de CSS escrito por Lola Cárdenas, una Guía de Estilo, y muchas más cosas.

Hay una lista muy interesante de recursos de diseño y desarrollo web en la página web del Sidar, incluido el manual de CSS de Carlos Benavidez, que recomiendo a todo el mundo.

Y por supuesto, si queréis un buen tutorial de diseño web en castellano, ¡os recomendamos el nuestro!

Tutoriales de HTML en inglés

En realidad, tampoco son muchos los tutoriales en inglés que os puedo recomendar. Hay uno que está a años luz de todos los demás, que es el tutorial de HTML de Stephanos Piperouglou. Todos los que sepáis inglés os tenéis que pasar por allí.

Es tan bueno que no voy a recomendar más.

Tutoriales y referencias de CSS en inglés

El punto de partida para buscar referencias sobre CSS es la página de estilo del W3C, que tiene montones de enlaces. Entre ellos los siguientes, que son muy recomendables:

  • The CSS Pointers Group
  • CSS School
  • Web Design Group
  • HTML Goodies
  • HTMLCenter
  • web.blazonry
  • Westciv

Las siguientes páginas contienen información y ejemplos de técnicas de aplicación de CSS, y en ellas podréis encontrar enlaces a muchas otras páginas interesantes:

  • BrainJar.com. Algunas partes de esta web han sido traducidas al castellano y están disponibles en kusor.net.
  • CSS Layout Techniques de glish.com.
  • The Layout Reservoir de BlueRobot.
  • The Noodle Incident.

Esta es una selección de sitios web interesantes sobre maquetación en CSS dada por Pascal Chevrel en un hilo de es.comp.infosistemas.www.paginas-web:

  • Artículos de Stopdesign
  • La página de Eric Meyer
  • Cinnamon Interactive es una compañía de diseño web y en su página hablan sobre rediseño de páginas con CSS (la página está en holandés).
  • A List Apart
  • Clagnut, la página personal de Richard Rutter, diseñador web.
  • CYBERcodeur.net, un blog sobre diseño web según los estándares (en francés).
  • Un tutorial sobre reconstrucción de páginas web con CSS por Christopher Schmitt.
